# Тоҷрасон Курйер

© 2024 Тоҷрасон. Ҳамаи ҳуқуқҳо маҳфузанд.

Қисми фронтенди лоиҳаи "Тоҷрасон" барои **куриерҳо**. Ин барнома ба курйерҳо имкон медиҳад, ки фармоишҳои дастрасро бубинанд, қабул кунанд, пайгирӣ намоянд ва даромади худро назорат кунанд.

## 📋 Технологияҳо

- **React 18** - Китобхонаи асосӣ барои сохтани UI
- **TypeScript** - Забони асосии барномасозӣ
- **Vite** - Воситаи сохтмон ва таҳия
- **Redux Toolkit** - Идоракунии ҳолати глобалӣ
- **React Router** - Масиргузории саҳифаҳо
- **React Hook Form + Zod** - Кор бо формаҳо ва валидатсия
- **Axios** - Дархостҳои HTTP ба API
- **Socket.io** - Пайвасти вақти воқеӣ (WebSocket)
- **Leaflet** - Намоиши харитаҳо ва масирҳо
- **Chart.js** - Графикҳо ва диаграммаҳо барои омор
- **Framer Motion** - Аниматсияҳо
- **Tailwind CSS** - Услубгузории зуд
